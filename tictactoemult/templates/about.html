TECHNICAL

Tic Tac Toe Multiplayer is a website built with a Python Django backend, a MySQL database, vanilla CSS and vanilla JavaScript and some jQuery for certain animations. The website also uses the Daphne WebSocket for real time communication in matchmaking and matches.

The code and other details are available on the GitHub repo for the project.
